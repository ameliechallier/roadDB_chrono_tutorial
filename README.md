This tutorial introduces how to deal with large chronological datasets using the roadDB package.
#
This tutorial is for radiocarbon dates only. However, note that it's possible to retrieve any kind of chronological data from the ROAD database by filtering by dating_method.

Radiocarbon dates can be calibrated using the rcarbon package in combination with roadDB. Radiocarbon ages to be used in age modelling software should remain uncalibrated.

This tutorial also creates a .csv file that can be used in the Chronomodel software: Lanos Ph. and Dufresne Ph. (2024). 'ChronoModel version 3.0: Software for Chronological Modelling of Archaeological Data using Bayesian Statistics'. 
Available at https://chronomodel.com/

Chronomodel should be downloaded independently using the provided link.
